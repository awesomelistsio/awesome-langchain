> A curated list of resources, tools, examples, and projects using [LangChain](https://www.langchain.com/), the framework for developing applications powered by language models.

LangChain is a powerful and extensible framework for building applications with large language models (LLMs), integrating tools, memory, chains, agents, and data sources.

## Official Resources

- **[LangChain Website](https://www.langchain.com/)** – Official site with docs, guides, and examples.
- **[LangChain GitHub](https://github.com/langchain-ai/langchain)** – Main open-source repo.
- **[LangChain Docs](https://docs.langchain.com/)** – Extensive documentation and API references.
- **[LangChainJS](https://github.com/langchain-ai/langchainjs)** – JavaScript/TypeScript version of LangChain.
- **[LangServe](https://github.com/langchain-ai/langserve)** – Tool for deploying LangChain apps as REST APIs.

## Learning Resources

- **[LangChain YouTube Channel](https://www.youtube.com/@LangChain)** – Tutorials, demos, and walkthroughs.
- **[LangChain Cookbook](https://github.com/hwchase17/langchain-cookbook)** – Recipes and guides for using LangChain effectively.
- **[LangChain with OpenAI](https://blog.langchain.dev/langchain-openai/)** – Guide on integrating LangChain with OpenAI models.
- **[LangChain Bootcamp (DeepLearning.AI)](https://www.deeplearning.ai/short-courses/langchain/)** – Free short course for LangChain.

## Community Projects & Examples

- **[LangChainHub](https://github.com/langchain-ai/langchainhub)** – Community-contributed chains, agents, and prompts.
- **[AgentGPT](https://github.com/reworkd/AgentGPT)** – Autonomous agents built using LangChain.
- **[FlowiseAI](https://github.com/FlowiseAI/Flowise)** – Low-code tool to build LLM flows using LangChain.
- **[GPT Researcher](https://github.com/assafelovic/gpt-researcher)** – Fully automated agent that writes research reports using LangChain.

## Templates & Boilerplates

- **[LangChain Chat Template](https://github.com/kyrolabs/langchain-chat-template)** – Next.js + LangChain + Pinecone starter.
- **[LangChain Next.js Starter](https://github.com/zahidkhawaja/langchain-nextjs-template)** – AI SaaS boilerplate using LangChain and OpenAI.
- **[LangChain + Supabase Starter](https://github.com/mayooear/langchain-supabase-chatbot)** – Build chatbots with LangChain and Supabase.

## Integrations

- **[Pinecone](https://www.pinecone.io/)** – Vector database used in many LangChain retrieval apps.
- **[Weaviate](https://weaviate.io/)** – Semantic vector search engine used with LangChain.
- **[Chroma](https://www.trychroma.com/)** – Open-source embedding database for LangChain.
- **[LlamaIndex](https://www.llamaindex.ai/)** – Integrates well with LangChain for connecting LLMs to external data.

## Tools & Extensions

- **[LangGraph](https://github.com/langchain-ai/langgraph)** – Add stateful and graph-based reasoning to LangChain agents.
- **[LangSmith](https://www.langchain.com/langsmith)** – LangChain’s debugging, tracing, and evaluation tool.
- **[Griptape](https://github.com/griptape-ai/griptape)** – Alternative LLM framework that can integrate with LangChain ideas.

## Production Tools & Deployment

- **[LangServe](https://github.com/langchain-ai/langserve)** – Package and deploy LangChain applications as APIs.
- **[Helicone](https://www.helicone.ai/)** – Logging and monitoring for OpenAI & LangChain usage.
- **[Modal](https://modal.com/)** – Serverless deployment for LangChain-powered apps.

## Open Source Projects

- **[Autogen](https://github.com/microsoft/autogen)** – Microsoft’s multi-agent framework (not LangChain-exclusive, but compatible).
- **[ChatLangChain](https://github.com/hwchase17/chat-langchain)** – Demo chatbot using LangChain and Streamlit.
- **[BabyAGI](https://github.com/yoheinakajima/babyagi)** – Simple task management agent powered by LangChain.
